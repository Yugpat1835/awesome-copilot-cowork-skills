# Persona template

Copy this file to create a new reviewer persona for the board. Name it `persona-<role>.md` (kebab-case role, for example `persona-general-counsel.md`) and keep it in this `references/` folder. The review board convenes any persona file matching that pattern when you ask for it by role name.

Two rules before you start:

- **Archetypes only.** A persona is a role, never a real, named individual. Build it from what the role universally owns and probes, not from any person's public profile. If you privately tune a persona to your own organisation, keep that file in your OneDrive, not in a public repository.
- **Fill every section.** The red-team validation stage depends on KNOWN BLIND SPOTS being honest; a persona without declared blind spots gets unearned authority in the cross-comment round.
