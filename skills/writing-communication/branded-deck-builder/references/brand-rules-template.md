# Brand rules (template)

How to use this file: copy it to `brand-rules.md` in this same `references` folder, then replace every value in [square brackets] with your organisation's real rules. Keep the section headings exactly as they are: the skill reads them by name. Delete any example rows you do not need, and add rows freely. If a section genuinely does not apply, write "Not applicable" under it rather than deleting the heading.
