## 📁 What is this?

This repository contains a collection of 31 automated skills designed for Microsoft 365 Copilot Cowork. You can use these tools to speed up your daily work tasks without needing special technical skills or administrative permissions. These skills function through simple folder-drop automation. When you move a folder into your OneDrive, the system triggers the specific workflow. 

This collection includes 16 pre-built workflows and a 16-persona executive review board. These tools help you organize, analyze, and process data using AI. You can also use these same skill files within Claude Code for broader project management needs. This system works entirely within your existing Microsoft 365 environment.

## 📋 System Requirements

To use these skills, ensure your computer meets these basic requirements:

*   Operating System: Windows 10 or Windows 11.
*   Software: Microsoft 365 subscription with Copilot enabled.
*   Storage: Microsoft OneDrive active and syncing with your device.
*   Account: A standard work or school Microsoft account.
*   Permission: No special administrator access is necessary for your computer.

You do not need to install any proprietary software packages or complex coding environments. The system relies on the native integration between OneDrive and Microsoft 365.

## ⚙️ Using the Workflow Tools

The collection includes 16 specialized workflows. Each workflow targets a specific business requirement, such as document summarization, email drafting, or project scheduling.

To activate a workflow, follow these steps:

1.  Open your OneDrive folder on your Windows computer.
2.  Locate the folder containing the specific skill you wish to use.
3.  Drag and drop your target documents into the designated folder.
4.  Microsoft 365 Copilot detects the new files and initiates the skill logic.
5.  Check your OneDrive for the updated or generated output files.

You can move multiple files at once. The system processes them in sequence based on the workflow instructions contained within each SKILL.md file.

## 💡 Troubleshooting

If a skill fails to trigger, check these common items:

*   Connection: Ensure your computer has a stable internet connection.
*   Sync Status: Look at the OneDrive icon in your Windows taskbar. Ensure it shows as "Up to date."
*   Permissions: Verify you have edit access to the specific OneDrive folder where you placed the files.
*   File Format: Ensure your files use standard formats like .docx, .pdf, or .txt.
*   Placement: Confirm the files sit inside the specific skill folder rather than the root directory.

If you encounter persistent issues, verify that your Microsoft 365 Copilot license remains active. You can check your subscription status through the Microsoft 365 web portal in your browser.
